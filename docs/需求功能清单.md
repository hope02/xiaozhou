# 小舟浏览器（鸿蒙）需求功能清单

## 1. 产品目标
- 产品名称：小舟
- 产品形态：鸿蒙（HarmonyOS）浏览器 App
- 核心目标：页面简洁大方、交互流畅，满足日常网页浏览需求
- 差异化能力：支持通过 WebDAV 同步书签，实现多设备数据一致

## 2. 目标用户与使用场景
- 日常信息浏览用户：新闻、搜索、学习资料、社区内容访问
- 轻度效率用户：收藏常用站点，快速访问高频页面
- 多设备用户：希望在不同设备间同步书签数据

## 3. 产品设计原则
- 简洁大方：界面元素克制，避免过度装饰和复杂层级
- 易用优先：高频操作一步可达（搜索、前进后退、收藏、历史）
- 稳定可靠：加载稳定、崩溃率低、同步可恢复
- 隐私透明：权限最小化、数据去向可说明、可手动清理

## 4. 核心功能清单

### 4.1 浏览能力（MVP 必须）
- 地址栏输入：支持 URL 与关键词输入
- 搜索引擎：默认搜索引擎配置（可扩展）
- 页面打开与渲染：基础网页加载、重定向处理、HTTPS 访问
- 标签页管理：新建、关闭、切换标签页
- 导航控制：前进、后退、刷新、停止加载
- 外链处理：支持系统级 Scheme 跳转（如 tel、mailto）

### 4.2 书签能力（MVP 必须）
- 新增书签：从当前页面一键收藏
- 书签管理：查看、编辑、删除、排序
- 书签分类：支持文件夹管理（一级目录优先）
- 快速访问：主页展示常用书签入口

### 4.3 历史记录（MVP 建议）
- 浏览历史自动记录
- 历史检索（按标题 / URL）
- 单条删除与一键清空

### 4.4 页面与阅读体验（MVP 建议）
- 简洁主页：搜索框 + 常用书签
- 夜间模式（跟随系统）
- 字体缩放（基础级）
- 横竖屏适配

### 4.5 下载与基础文件处理（V1）
- 常见文件下载（图片、文档、压缩包）
- 下载列表查看（进行中 / 已完成 / 失败）
- 下载失败重试

## 5. WebDAV 书签同步需求（重点）

### 5.1 账号与连接配置（MVP 必须）
- 配置项：服务器地址、用户名、密码、同步目录
- 连接测试：保存前执行连通性验证
- 错误提示：对认证失败、路径不存在、网络异常给出明确文案

### 5.2 同步机制（MVP 必须）
- 同步对象：仅书签与书签文件夹
- 同步方式：双向增量同步（本地 <-> WebDAV）
- 同步时机：
  - 手动触发同步
  - 应用启动后自动同步（可开关）
  - 应用进入后台前尝试同步（可开关）
- 数据格式：统一 JSON 文件（建议附版本号字段）

### 5.3 冲突处理（MVP 必须）
- 冲突识别：同一书签在不同端被修改
- 基础策略：按最后修改时间优先（LWW）
- 用户可选策略（V1）：保留本地 / 保留云端 / 合并后确认
- 冲突日志：记录冲突条目，便于排查

### 5.4 安全与隐私（MVP 必须）
- 凭据本地安全存储（系统安全存储能力）
- 传输要求：仅允许 HTTPS WebDAV（MVP）
- 本地缓存加密（V1 可选）

### 5.5 异常恢复（MVP 建议）
- 断网重试：指数退避重试策略
- 部分失败可回滚或重放
- 保留最近一次可用快照用于恢复

## 6. 非功能需求

### 6.1 性能
- 冷启动时间：目标 <= 2.5 秒（中端设备）
- 首屏可交互时间：目标 <= 1.5 秒（常见页面）
- 滑动与切换流畅，无明显卡顿

### 6.2 稳定性
- 崩溃率控制：核心路径崩溃率低于目标阈值
- 弱网场景可用：加载超时可重试，页面状态可感知

### 6.3 可维护性
- 模块化设计：浏览内核层、数据层、同步层、UI 层分离
- 日志分级：调试日志与线上日志区分
- 关键链路埋点：页面加载、书签变更、同步成功率

## 7. 权限与合规
- 网络访问权限
- 文件读写（仅在下载和导入导出场景申请）
- 隐私说明：书签同步范围、存储位置、用户可删除权

## 8. 验收标准（建议）
- 能完成日常浏览：搜索、访问、标签管理、历史管理
- 书签完整可用：新增、编辑、删除、分类正常
- WebDAV 同步稳定：首次全量 + 日常增量同步成功率达标
- 异常处理可用：断网、鉴权失败、冲突场景有明确反馈
- UI 风格统一：整体简洁大方，关键操作路径清晰

## 9. 版本规划（建议）

### MVP（首发）
- 基础浏览（地址栏、标签、导航）
- 书签与历史
- WebDAV 书签同步（手动 + 启动自动）
- 基础设置（搜索引擎、外观跟随系统）

### V1.1
- 下载管理完善
- 同步冲突可视化处理
- 同步稳定性增强（重试、快照恢复）

### V1.2
- 阅读模式优化
- 个性化主页（常用站点编辑）
- 数据导入导出（书签）
